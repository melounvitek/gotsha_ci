your local CI

Sign off your commits locally — and prove it.

Run tests & linters on your machine, store results as Git notes, and let your PR show the proof.

Install How it works

1) Run locally

Gotsha runs your test suite and linters using your config. Fast, offline, no queue.

2) Record the SHA

Results are attached as Git notes to the commit SHA.

3) Show in your PR

Push as usual — the note follows your commit, making the proof visible to reviewers.

Install

# RubyGems (example)
gem install gotsha

# Or add to your Gemfile
gem 'gotsha'
